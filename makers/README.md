# 만든 사람들 아바타 이미지

이 폴더에 아래 파일명으로 이미지를 넣으면 "만든 사람들" 화면 카드/챗봇에 자동 반영됩니다.
파일이 없으면 이모지 아바타로 대체됩니다(앱은 그대로 동작).

| 파일명 | 인물 | 비고 |
|---|---|---|
| `jeongrae.png` | 정래 (팀장) | 헤드폰+안경 일러스트 |
| `jihoon.png`  | 지훈 (Main Builder) | 안경 없는 남성 일러스트 |
| `minho.png`   | 민호 (Data Owner) | 검은 뿔테 안경 남성 일러스트 |
| `gyeongmok.png` | 경목 (QA Owner) | 갈색 자켓 남성 일러스트 |
| `heeyeon.png` | 희연 (Design Owner) | 빨간 체크셔츠 포니테일 |
| `youngin.png` | 영인 (헤더코치) | 노란 리본 일러스트 |

- 정사각형(1:1) 이미지 권장. png/jpg 가능(코드의 확장자는 .png 기준이라 jpg면 Makers.tsx의 경로 확장자만 바꾸면 됩니다).
- 정적 폴더(`public/`)라 빌드 시 그대로 `/makers/...` 경로로 서빙됩니다.
